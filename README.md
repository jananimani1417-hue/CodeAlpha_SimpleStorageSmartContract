# CodeAlpha_SimpleStorageSmartContract
The contract demonstrates basic state variable management and smart contract interaction.
Project Overview

This project is a simple storage smart contract built using Solidity. It stores an integer value and provides functions to increase, decrease, and view the value.

Features
Store an integer value
Increment value by 1
Decrement value by 1
Retrieve stored value
Tested on Remix IDE
Technologies Used
Solidity
Ethereum
Remix IDE

Outcome

Successfully deployed and tested the contract with increment and decrement operations.
